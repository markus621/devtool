```bash
go get -u github.com/markus621/devtool/cmd/...
```

# commands

```bash
golang

Usage:
  devtool go [command]

Available Commands:
  env         list system environment
  i           installing golang for linux

Flags:
  -h, --help   help for go

Use "devtool go [command] --help" for more information about a command.
```
